# Development Log

## Project Overview
Personal website built with Jekyll, using the Moonwalk theme as a base.

## Framework & Structure
- **Framework**: Jekyll
- **Theme**: Moonwalk
- **Language**: English
- **Deployment**: GitHub Pages

## Directory Structure
```
.
├── _includes/          # Reusable components
├── _layouts/          # Page templates
├── _posts/            # Blog posts
├── _sass/             # SCSS styles
├── assets/            # Static assets
│   ├── css/          # CSS files
│   ├── js/           # JavaScript files
│   ├── images/       # Images
│   └── logos/        # Company logos
├── _config.yml        # Site configuration
└── [various].md       # Content pages
```

## Pages & Features

### 1. Home Page (`index.markdown`)

- Navigation menu

### 2. Blog Page (`blog.md`)
